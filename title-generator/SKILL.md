# 短视频爆款标题生成器

## 技能信息

- **技能名称**：短视频爆款标题生成器
- **版本**：v1.0
- **作者**：Agent Assistant
- **适用平台**：抖音、快手、小红书、视频号、B站

## 功能描述

一款专为短视频创作者打造的智能标题生成工具。基于平台算法偏好和爆款内容规律，生成具有高点击率、高传播力的短视频标题。支持多种标题类型，满足不同内容风格需求。

## 核心能力

### 支持的标题类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 悬念式 | 设置悬念，引发好奇 | 知识科普、剧情类 |
| 数字式 | 具体数据，吸引眼球 | 教程、干货、合集 |
| 疑问式 | 提问互动，引发思考 | 观点输出、讨论类 |
| 反转式 | 出人意料，冲击感强 | 剧情、搞笑、测评 |
| 情感式 | 引发共鸣，触动内心 | 故事、生活、励志 |

## 使用方法

### 基本指令

```
生成短视频标题
主题：你的视频内容
类型：悬念式/数字式/疑问式/反转式/情感式
平台：抖音/快手/小红书/视频号
数量：3-5条
```

### 快捷指令

- `@标题 悬念 [内容描述]` - 生成悬念式标题
- `@标题 数字 [内容描述]` - 生成数字式标题
- `@标题 疑问 [内容描述]` - 生成疑问式标题
- `@标题 反转 [内容描述]` - 生成反转式标题
- `@标题 情感 [内容描述]` - 生成情感式标题
- `@标题 爆款 [内容描述]` - 全类型综合生成

### 示例

**输入**：
```
主题：分享3个让衣柜整洁的收纳技巧
类型：数字式
平台：抖音
数量：5条
```

**输出**：
```
【短视频爆款标题 - 数字式】

1. 衣柜乱糟糟？3个收纳技巧让空间翻倍！
2. 学会这3步，衣柜瞬间整洁如新！
3. 难怪衣服找不到！3个收纳误区你中了几个？
4. 小空间大容量！3个衣柜收纳神技巧
5. 衣柜收纳天花板！学会这3招就够了
```

## 标题生成规则

### 平台适配

- **抖音**：短平快，情绪强烈，前3字抓眼球
- **快手**：接地气，有烟火气，强调真实感
- **小红书**：精致感，有调性，突出价值感
- **视频号**：稳重感，有深度，正能量导向
- **B站**：年轻化，有梗，可玩性高

### 爆款公式

```
爆款标题 = 痛点/痒点 + 数字/悬念 + 平台关键词
```

### 禁忌事项

- ❌ 标题党过度（与内容严重不符）
- ❌ 使用绝对化词汇（必须、一定、绝不）
- ❌ 敏感词/违规词
- ❌ 过度夸张（全球第一、震惊等）
- ❌ 标题过长（建议15-25字）

### 加分技巧

- ✅ 使用阿拉伯数字（更醒目）
- ✅ 巧用标点符号（！？：）
- ✅ 植入平台热词
- ✅ 制造紧迫感/稀缺感
- ✅ 引发好奇心或共鸣
