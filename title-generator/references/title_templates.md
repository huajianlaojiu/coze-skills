## 六、爆款标题万能公式

### 公式清单

```
公式1：数字 + 痛点 + 解决方案
示例：5个让衣柜翻倍的收纳技巧

公式2：悬念 + 身份 + 行动
示例：月薪3000的人，是怎么攒下钱的

公式3：对比 + 反差 + 结果
示例：从土肥圆到女神，她只做了这件事

公式4：提问 + 痛点 + 答案
示例：皮肤差怎么办？试试这个急救方法

公式5：身份 + 场景 + 情感
示例：独居女孩的深夜食堂，简单却治愈

公式6：热点 + 观点 + 争议
示例：XX塌房了，我却想为他说句话

公式7：结果 + 过程 + 方法
示例：30天练出马甲线，我是这么做的
```

### 标题优化技巧

1. **数字替换**：阿拉伯数字 > 中文数字
2. **符号运用**：`「」『』「」『』` 引号、`！？！` 感叹疑问
3. **词汇升级**：
   - "不错" → "绝了/太可了/真香"
   - "很多" → "爆了/炸裂/逆天"
   - "厉害" → "天花板/神仙/ yyds"
4. **句式选择**：陈述句 vs 疑问句 vs 感叹句
5. **长度把控**：抖音12-20字，小红书15-25字
