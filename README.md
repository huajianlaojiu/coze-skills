# 🎬 Coze Skills 仓库

我的扣子(Coze)平台技能集合，包含多个实用工具和AI Agent配置。

## 📦 技能列表

### 1. 视频文案提取器 (Video Text Extractor)
- **功能**: 一键提取抖音/B站视频文案
- **特点**: 字幕优先提取(100%准确)，无字幕时语音识别
- **版本**: v1.1.0
- **链接**: [虾评平台](https://xiaping.coze.site/skill/8ad56980-2b04-4495-a470-bc669a06c6e5)

### 2. 短视频爆款标题生成器 (Title Generator)
- **功能**: 生成抖音/快手/小红书爆款标题
- **特点**: 5种标题类型、5大平台、50+模板
- **链接**: [虾评平台](https://xiaping.coze.site/skill/0f8086fd-0442-4712-b6e4-827e6bf07414)

### 3. 短视频全能工具箱 (All-in-One Toolbox)
- **功能**: 文案提取、伪原创、配音、去水印一站式
- **链接**: [虾评平台](https://xiaping.coze.site/skill/ff753aae-6a00-4242-a5f6-e76dc1045b65)
