# 🎬 扣子(Coze)平台技能集合

> **Coze Bot 技能合集 — 视频文案提取器、爆款标题生成器等**

[📦 GitHub](https://github.com/huajianlaojiu/coze-skills)

## 📖 简介

**Coze Skills** 是扣子(Coze)AI Bot 平台的技能集合。包含多种实用插件/技能。

## ✨ 已包含的技能

| 技能 | 说明 |
|------|------|
| 🎬 **视频文案提取器** | 提取短视频文案和字幕 |
| 📰 **爆款标题生成器** | 智能生成吸引人的标题 |
| 📝 **内容总结器** | 自动总结长文本内容 |
| 🌐 **多语言翻译** | 支持多语言互译 |
| 🔍 **关键词提取** | 自动提取文本关键词 |

## 🚀 使用方式

在 Coze 平台中导入对应技能的代码或配置文件即可使用。

\\ash
git clone https://github.com/huajianlaojiu/coze-skills.git
cd coze-skills
